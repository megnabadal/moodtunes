# 🎵 MoodTunes

> A mood-based music recommender that suggests songs based on how you're feeling right now.

![MoodTunes Preview](https://img.shields.io/badge/Made%20with-HTML%20%7C%20CSS%20%7C%20JS-blue?style=flat-square)
![Status](https://img.shields.io/badge/Status-Live-brightgreen?style=flat-square)

## ✨ What it does

MoodTunes lets you pick your current mood and instantly generates a curated playlist of song recommendations — complete with genre tags, BPM vibe, and Spotify search links. No API keys, no sign-up. Just your mood and your music.

## 🚀 Features

- 6 mood categories: Happy, Sad, Energetic, Chill, Focused, Romantic
- Instant song recommendations with artist names and genres
- Spotify search links for every song
- Smooth animations and a beautiful dark UI
- 100% client-side — works offline after first load

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid)
- Vanilla JavaScript (no frameworks, no libraries)
